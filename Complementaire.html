Complémentaire d’un automate — exemples type annales

Idée principale : si l’automate est déterministe et complet, le complémentaire se construit en gardant les mêmes transitions et en inversant seulement les états finaux.

Règle examen :
Pour construire le complémentaire de L(A), il faut d’abord que l’automate soit déterministe et complet. Ensuite, on garde les mêmes états, le même état initial, les mêmes transitions, et on remplace les états finaux F par Q \ F.

Exemple 1 — Mots qui se terminent par b

Alphabet : A = {a,b}.
Automate original : reconnaît les mots qui se terminent par b.
Complémentaire : reconnaît les mots qui ne se terminent pas par b.

Automate original

q0 : ne se termine pas par b.
q1 : se termine par b.
État final : q1.

Automate complémentaire

On inverse les états finaux : q0 devient final, q1 ne l’est plus.
Acceptés : ε, a, ba, aba.
Refusés : b, ab, bb.

Exemple 2 — Automate non complet : il faut ajouter un état puits

Alphabet : A = {a,b}.
Automate original non complet : reconnaît a*, c’est-à-dire les mots ne contenant que des a.
Pour prendre le complémentaire, on complète d’abord l’automate avec un état puits P.

Automate complété pour a*

q0 final : aucun b lu.
P non final : on a déjà lu un b.
L’automate est maintenant complet.

Complémentaire : mots contenant au moins un b

On inverse : q0 devient non final, P devient final.
Acceptés : b, ab, ba, aab.
Refusés : ε, a, aa.

Exemple 3 — Nombre de b divisible par 3

Alphabet : A = {a,b}.
Automate original : reconnaît les mots dont le nombre de b est divisible par 3.
Complémentaire : reconnaît les mots dont le nombre de b n’est pas divisible par 3.

Automate original

Final : q0 seulement.
q0 : reste 0 modulo 3 ; q1 : reste 1 ; q2 : reste 2.

Automate complémentaire

Final : q1 et q2.
Acceptés : mots avec 1, 2, 4, 5, ... lettres b.
Refusés : mots avec 0, 3, 6, ... lettres b.

Exemple 4 — Automate à 4 états : mots contenant aaa

Alphabet : A = {a,b}.
Automate original : reconnaît les mots qui contiennent trois a consécutifs, donc le facteur aaa.
Complémentaire : reconnaît les mots qui ne contiennent pas le facteur aaa.

Automate original : contient aaa

q0 : 0 a consécutif à la fin.
q1 : on vient de lire 1 a.
q2 : on vient de lire 2 a consécutifs.
q3 : on a déjà vu aaa. Final.

Complémentaire : ne contient pas aaa

On inverse les états finaux : q0, q1, q2 deviennent finaux ; q3 ne l’est plus.
Acceptés : ε, a, aa, aaba.
Refusés : aaa, baaa, aaab.
À retenir : le complémentaire ne change pas les flèches. Il change seulement les doubles cercles, mais uniquement après avoir vérifié que l’automate est déterministe et complet.
