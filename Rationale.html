Expressions rationnelles — درس کامل + سؤال‌های Annales

هدف: بتوانی از روی یک expression rationnelle زبان را توضیح بدهی، و از روی توضیح یک زبان، expression بسازی. آخر فایل چند سؤال annales-style و exact annales با حل مرحله‌به‌مرحله آمده است.

۱) تعریف خیلی ساده

expression rationnelle یا expression régulière یک فرمول است که یک زبان را توصیف می‌کند. مثلاً به جای گفتن «همه کلماتی که با ab تمام می‌شوند»، می‌نویسیم:

A*ab

E + Fیا E یا Fa+b={a,b}

دقت خیلی مهم: a+b در expression rationnelle معمولاً یعنی a ou b، اما a+ در بعضی notationها یعنی «یک یا چند a». پس فاصله و contexte مهم است.

۵) سؤال Annale 2026 — expression rationnelle

Question exacte visible dans le fichier annales :
On se place sur l'alphabet réduit à une seule lettre A = {a}.
Soit L le langage sur A contenant les mots de longueur n, pour tout n vérifiant n = 2 + 3 × k, pour k ∈ N.
a. (1 point) Donner une expression rationnelle associée à L.
b. (1 point) Dessiner un automate reconnaissant L.

حل مرحله‌به‌مرحله

Étape 1. چون n = 2 + 3k، اول باید دقیقاً دو a داشته باشیم:

aa

Étape 2. بعد می‌توانیم هر بار سه a اضافه کنیم:

aaa

Étape 3. چون k می‌تواند صفر، یک، دو، ... باشد، سه‌تایی‌ها با * می‌آیند:

aa(aaa)*

Réponse :

L = aa(aaa)*

مثال‌ها: aa طول ۲، aaaaa طول ۵، aaaaaaaa طول ۸.

Automate correspondant

ایدهٔ automate: هر a طول را یکی زیاد می‌کند. ما طول را modulo 3 حساب می‌کنیم. حالت final وقتی است که طول ≡ 2 mod 3 باشد.

روش معادلات زبان

برای هر état یک متغیر می‌گذاریم:

X0, X1, X2, X3

Xi یعنی زبان کلماتی که اگر از état i شروع کنیم، ما را به final می‌رسانند.

ÉtatÉquation
X0X0 = aX1 + bX2
X1X1 = ε + aX1 + bX3
X2X2 = aX3
X3X3 = (a+b)X0

حل خلاصه با Arden

از X1 = ε + aX1 + bX3 داریم:

X1 = a*(ε + bX3)

جایگذاری:

X3 = (a+b)X0

X2 = a(a+b)X0

حالا:

X0 = aX1 + bX2

X0 = aa* + (aa*b + ba)(a+b)X0

با Arden:

X0 = ((aa*b + ba)(a+b))* aa*

پس یک expression rationnelle برای L(A):

((aa*b + ba)(a+b))*aa*

۷) سؤال Annale 2023 — توجه به زبان غیرقابل تشخیص

در فایل annales، سؤال 2023 شامل L2 = {a^{2n}b^{2n+1} | n ∈ N} و بحث recognisable / non-reconnaissable است.
نکتهٔ مهم: هر زبان الزاماً expression rationnelle ندارد. اگر زبان non reconnaissable باشد، expression rationnelle هم برای آن وجود ندارد.

مثلاً:

{a^{2n}b^{2n+1} | n ∈ N}

اینجا تعداد aها و bها به هم وابسته‌اند. اتومات finite نمی‌تواند هر عدد n را نگه دارد. پس نباید برایش دنبال expression rationnelle ساده بگردی.

۹) یک سؤال آخر برای تو

این را حل کن و عکس بفرست:

Soit A = {a,b}. Donner une expression rationnelle pour chacun des langages suivants :
  1. Les mots qui se terminent par ba.
  2. Les mots qui contiennent exactement deux b.
  3. Les mots qui commencent par a et contiennent au moins un b.
  4. Les mots de longueur paire.
جواب کامل باید فقط expression نباشد؛ برای هر کدام ۲ کلمه accepted و ۲ کلمه refused هم بنویس.

۱۰) خلاصه exam-ready

Une expression rationnelle décrit un langage. Les opérations principales sont l'union + ou |, la concaténation et l'étoile de Kleene *. Pour passer d'un langage à une expression, on traduit la propriété du langage en blocs : début, milieu, fin, nombre exact de lettres, répétition. Pour passer d'un automate à une expression, on peut écrire des équations de langages et utiliser le lemme d'Arden.
