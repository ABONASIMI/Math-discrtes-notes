LU2IN005 — TD2 Part 5: Bounds
minorant / majorant / inf / sup

0. Real goal first

In this part, you have a big ordered set E and a subset X ⊆ E.

E = big set
X = subset of E

Bounds are searched in E, not only inside X.
ConceptWhere must the answer be?Goal
min(X), max(X)Inside XElement of X below/above all X
minorant, majorantInside EElement of E below/above all X
inf(X), sup(X)Inside EBest lower/upper bound

1. Definitions

minorant = lower bound

m est un minorant de X
⇔ m ∈ E et ∀x ∈ X, m ≤ x

Real meaning: m is below every element of X.

majorant = upper bound

M est un majorant de X
⇔ M ∈ E et ∀x ∈ X, x ≤ M

Real meaning: M is above every element of X.

inf(X)

inf(X) = le plus grand des minorants de X

sup(X)

sup(X) = le plus petit des majorants de X

2. Universal algorithm

To find minorants:
1. Take every element e in E.
2. Test: e ≤ every element of X?
3. If yes, e is a minorant.

To find majorants:
1. Take every element e in E.
2. Test: every element of X ≤ e?
3. If yes, e is a majorant.

inf(X) = biggest minorant
sup(X) = smallest majorant

3. Example A — normal order

E = {0,1,2,3,4,5}
X = {2,3}
order = normal ≤
Yellow border = elements of X. Green = lower bounds. Blue = upper bounds.
minorants(X) = {0,1,2}
majorants(X) = {3,4,5}
inf(X) = 2
sup(X) = 3

4. Example B — divisibility order

E = {1,2,3,6}
X = {2,3}
x ≤ y ⇔ x divise y

In divisibility order:

lower = divisor
upper = multiple
1 divides all → lower bound. 6 is divisible by all → upper bound.
minorants(X) = {1}
majorants(X) = {6}
inf(X) = 1
sup(X) = 6
Shortcut:
minorants = common divisors
majorants = common multiples inside E
inf = pgcd if it belongs to E
sup = ppcm if it belongs to E

5. Example C — bigger divisibility example

E = {1,2,3,4,6,12}
X = {4,6}
x ≤ y ⇔ x divise y
divisors of 4 in E = {1,2,4}
divisors of 6 in E = {1,2,3,6}
minorants(X) = common divisors = {1,2}

multiples of 4 in E = {4,12}
multiples of 6 in E = {6,12}
majorants(X) = common multiples = {12}

inf(X) = 2
sup(X) = 12

6. Example D — product order

E = ℕ × ℕ
X = {(1,2), (2,1)}
(a,b) ≤ (c,d) ⇔ a ≤ c et b ≤ d

In product order, compare both coordinates at the same time.

minorants:
(a,b) ≤ (1,2) and (a,b) ≤ (2,1)
So a ≤ 1 and b ≤ 1
If ℕ contains 0:
minorants = {(0,0),(0,1),(1,0),(1,1)}
inf(X) = (1,1)

majorants:
(1,2) ≤ (a,b) and (2,1) ≤ (a,b)
So a ≥ 2 and b ≥ 2
sup(X) = (2,2)

7. TD2 Hasse diagram style

V = {4,5,6}

sup(V) = 3

8. Common mistakes

MistakeCorrect idea
Bounds must be inside XNo. Bounds must be inside E.
inf = min alwaysNo. inf can exist even when min does not.
sup = max alwaysNo. sup can exist even when max does not.
Use normal order every timeNo. First translate the relation.
