۰. این فایل دقیقاً برای چیست؟

اینجا فقط minimal/maximal نیست. در هر مثال همهٔ چیزها را با هم پیدا می‌کنیم:

éléments minimaux
éléments maximaux
min(X)
max(X)
minorants(X)
majorants(X)
inf(X)
sup(X)

در تمام مثال‌ها رابطه این است:

x ≤ y ⇔ x divise y

یعنی:

پایین‌تر = diviseur = عددی که تقسیم می‌کند
بالاتر = multiple = عددی که بر آن عدد بخش‌پذیر است

۱. روش اصلی که باید حفظ کنی

minimal:
داخل X است و هیچ عضو دیگر X پایین‌تر از آن نیست.

maximal:
داخل X است و هیچ عضو دیگر X بالاتر از آن نیست.

min(X):
داخل X است و پایین همهٔ X است.

max(X):
داخل X است و بالای همهٔ X است.

minorant:
داخل E است، پایین همهٔ X است. لازم نیست داخل X باشد.

majorant:
داخل E است، بالای همهٔ X است. لازم نیست داخل X باشد.

inf(X):
بزرگ‌ترین minorant.

sup(X):
کوچک‌ترین majorant.

برای رابطهٔ تقسیم‌پذیری:

minorants(X) = diviseurs communs de tous les éléments de X dans E
majorants(X) = multiples communs de tous les éléments de X dans E
inf(X) = pgcd, si autorisé dans E
sup(X) = ppcm, si autorisé dans E

Example 1 — X has 3 elements

E = {1,2,3,6}
X = {2,3,6}
x ≤ y ⇔ x divise y

هدف این مثال: چند minimal داریم، minimum نداریم، اما maximum و sup داریم. همچنین inf بیرون X است.

Step 1 — éléments minimaux

2: no element of X divides 2 except 2 → minimal.
3: no element of X divides 3 except 3 → minimal.
6: 2 divides 6 and 3 divides 6 → not minimal.

Step 2 — éléments maximaux

2: 2 divides 6 → not maximal.
3: 3 divides 6 → not maximal.
6: 6 divides no other different element of X → maximal.

Step 3 — min(X) / max(X)

min(X):
Need one element inside X that divides all X.
2 does not divide 3.
3 does not divide 2.
So min(X) does not exist.

max(X):
2 divides 6 and 3 divides 6.
So max(X)=6.

Step 4 — minorants / majorants

minorants(X):
common divisors of 2,3,6 inside E.
Only 1 divides 2,3,6.
minorants(X) = {1}

majorants(X):
common multiples of 2,3,6 inside E.
6 is the only common multiple in E.
majorants(X) = {6}

Step 5 — inf / sup

inf(X) = greatest minorant = 1
sup(X) = smallest majorant = 6
éléments minimaux = {2,3}
éléments maximaux = {6}
min(X) = n’existe pas
max(X) = 6
minorants(X) = {1}
majorants(X) = {6}
inf(X) = 1
sup(X) = 6

Example 2 — X has 4 elements

E = {1,2,3,4,6,12}

هدف این مثال: یک minimum و یک maximum داریم. ولی minorants بیشتر از min هستند، چون 1 بیرون X هم minorant است.

Maximal:

4: divides 12 → not maximal.
6: divides 12 → not maximal.
12: divides no different element of X → maximal.
éléments maximaux = {12}

Step 2 — min / max

2 divides all X: 2|2, 2|4, 2|6, 2|12.
min(X)=2.

All X divide 12: 2|12, 4|12, 6|12, 12|12.
max(X)=12.

Step 3 — minorants / majorants

minorants(X):
common divisors of 2,4,6,12 in E:
{1,2}

majorants(X):
common multiples of 2,4,6,12 in E:
{12}

Step 4 — inf / sup

inf(X) = greatest minorant = 2
sup(X) = smallest majorant = 12
éléments minimaux = {2}
éléments maximaux = {12}
min(X) = 2
max(X) = 12
minorants(X) = {1,2}
majorants(X) = {12}
inf(X) = 2
sup(X) = 12

خلاصهٔ نهایی برای قبل امتحان

۱. اگر سؤال minimal خواست:
فقط داخل X نگاه کن.
اگر یک عضو دیگر X آن را تقسیم کند، minimal نیست.

۲. اگر سؤال maximal خواست:
فقط داخل X نگاه کن.
اگر آن عدد یک عضو دیگر X را تقسیم کند، maximal نیست.

۳. اگر min خواست:
یک عضو داخل X باید همهٔ X را تقسیم کند.

۴. اگر max خواست:
همهٔ اعضای X باید آن عضو را تقسیم کنند.

۵. اگر minorant خواست:
داخل E نگاه کن، نه فقط X.
عدد باید همهٔ X را تقسیم کند.

۶. اگر majorant خواست:
داخل E نگاه کن، نه فقط X.
همهٔ X باید آن عدد را تقسیم کنند.

۷. inf = بزرگ‌ترین minorant.
۸. sup = کوچک‌ترین majorant.
