Main goal: 1. Recognize the type of order. 2. Translate the order. 3. Apply the right tool.

1. Ordered set / order symbols

ایدهٔ واقعی: order یک قانون است برای گفتن اینکه کدام عنصر پایین‌تر یا بالاتر است.

(E, ≤)

یعنی مجموعهٔ E با رابطهٔ order.

x ≤ y

یعنی x پایین‌تر یا مساوی y است، ولی معنی دقیقش به relation بستگی دارد.

x < y ⇔ x ≤ y and x ≠ y
≤, <, ≼, ⪯, ⊑

این‌ها فقط اسم‌های مختلف برای relation هستند. همیشه تعریف سؤال را بخوان.

2. Types of order

ایدهٔ واقعی: باید بفهمی آیا هر دو عنصر قابل مقایسه‌اند یا نه.

ordre partiel = some elements may be non comparable ordre total = every two elements are comparable
x and y comparable ⇔ x ≤ y or y ≤ x x and y non comparable ⇔ x ≰ y and y ≰ x

برای antisymmetry:

si x ≤ y et y ≤ x alors x = y

3. Monotone / isomorphisme

ایدهٔ واقعی: تابع monotone ترتیب را حفظ می‌کند. isomorphisme یعنی دو order از نظر ساختار یکی هستند، فقط اسم عناصر فرق کرده است.

monotone: x ≤ y ⇒ f(x) ≤ f(y) isomorphisme: x ≤ y ⇔ f(x) ≤ f(y)

4. min / max

ایدهٔ واقعی: minimum یعنی global bottom داخل X. maximum یعنی global top داخل X.

m = min(X) ⇔ m ∈ X and ∀x ∈ X, m ≤ x M = max(X) ⇔ M ∈ X and ∀x ∈ X, x ≤ M

در divisibility:

X = {2,4,6,12} x ≤ y ⇔ x divise y min(X) = 2 max(X) = 12

5. Bounds: minorant / majorant / inf / sup

ایدهٔ واقعی: min/max باید داخل X باشند، اما minorant/majorant داخل E جستجو می‌شوند.

m is a minorant of X ⇔ m ∈ E and ∀x ∈ X, m ≤ x M is a majorant of X ⇔ M ∈ E and ∀x ∈ X, x ≤ M inf(X) = greatest minorant sup(X) = smallest majorant

6. minimal / maximal

ایدهٔ واقعی: minimal یعنی local bottom. minimum یعنی global bottom. maximal یعنی local top. maximum یعنی global top.

a minimal in X ⇔ a ∈ X and there is no x ∈ X with x < a a maximal in X ⇔ a ∈ X and there is no x ∈ X with a < x

خودش پایین همهٔ X است.

7. Hasse / divisibility / pgcd / ppcm

ایدهٔ واقعی: Hasse diagram ابزار دیدن order است. در divisibility پایین یعنی divisor و بالا یعنی multiple.

x | y means x divise y x ≤ y ⇔ x | y small = diviseur big = multiple

در Hasse diagram فقط linkهای مستقیم را می‌کشیم.

Hasse diagram: No reflexive links No transitive links Direct links only

در divisibility:

minorants = common divisors majorants = common multiples inf = pgcd sup = ppcm

8. Inclusion order / set operations

ایدهٔ واقعی: اینجا عناصر خودشان set هستند. پایین یعنی subset کوچک‌تر.

𝒫(E), ⊆ ∩, ∪ ⋂ Ai, ⋃ Ai E \ X
A ≤ B ⇔ A ⊆ B inf(A,B) = A ∩ B sup(A,B) = A ∪ B
A = {a,b} B = {b,c} A ∩ B = {b} A ∪ B = {a,b,c} inf(A,B) = {b} sup(A,B) = {a,b,c}

9. Words and word orders

ایدهٔ واقعی: اینجا با کلمه‌ها کار می‌کنیم و باید بتوانی کلمه‌ها را مقایسه کنی.

≤lex préfixe sous-mot |u| u ≡ v A*/≡ [u]
|u| = length of word u u préfixe de v ⇔ v starts with u u sous-mot de v ⇔ u can be obtained from v by deleting letters and keeping order ≤lex = dictionary order, compare from left to right
[u] = { v ∈ A* | v ≡ u } A*/≡ = set of all equivalence classes

10. Well-founded / sequences / product order

ایدهٔ واقعی: well-founded یعنی نمی‌شود تا بی‌نهایت پایین رفت. بالا رفتن بی‌نهایت مهم نیست.

ordre bien fondé chaîne descendante (en)n∈ℕ ℕ × ℕ (m1,m2) ≤ (n1,n2)
well-founded ⇔ no infinite descending chain well-founded ⇔ every non-empty subset has a minimal element
descending chain: e0 > e1 > e2 > e3 > ... (en)n∈ℕ = sequence e0, e1, e2, ...
ℕ × ℕ = pairs of natural numbers (m1,m2) ≤ (n1,n2) ⇔ m1 ≤ n1 and m2 ≤ n2
(ℕ, ≤) is well-founded: 5 > 4 > 3 > 2 > 1 > 0 ℤ is not well-founded: 0 > -1 > -2 > -3 > ...

Quiz with answers

این همان quiz کوتاهی است که انجام دادیم، با جواب درست.

Q1 — minimal

Un élément minimal de X est un élément de X qui n’a aucun élément de X strictement plus petit que lui. In divisibility: a is minimal if no other element of X divides a.

Q2 — minimal vs minimum

minimal = local bottom minimum = global bottom

Q3 — X = {2,4,6,12}

éléments minimaux = {2} éléments maximaux = {12} minimum = 2 maximum = 12

Q4 — divisibility bounds

minorants = diviseurs communs majorants = multiples communs inf = pgcd sup = ppcm

Q5 — X = {4,6}

E = {1,2,3,4,6,12} X = {4,6} minorants(X) = {1,2} majorants(X) = {12} inf(X) = 2 sup(X) = 12

Q6 — inclusion inf/sup

A ≤ B ⇔ A ⊆ B inf(A,B) = A ∩ B sup(A,B) = A ∪ B

Q7 — A={a,b}, B={b,c}

A ∩ B = {b} A ∪ B = {a,b,c} inf(A,B) = {b} sup(A,B) = {a,b,c}

Q8 — prefix / sous-mot

v = abcd u = bc bc is not a prefix of abcd. bc is a sous-mot of abcd.

Q9 — lexicographic

a < b < c abac

چون از چپ به راست: a=a، b=b، سپس a<b.

Q10 — well-founded

Un ordre est bien fondé si on ne peut pas descendre infiniment. (ℕ, ≤) est bien fondé.
