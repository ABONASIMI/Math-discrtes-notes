هدف: وقتی در معادلات زبان به چیزی مثل X = aX + b یا X = AX + B رسیدی، بتوانی سریع expression rationnelle نهایی را پیدا کنی.

۰) اول جواب سؤال تو: آیا a+b را می‌توانیم ab بنویسیم؟

خیر. a+b و ab اصلاً یکی نیستند.
Expressionمعنیزبان

در expression rationnelle، علامت + یعنی یا / union. اما چسبیدن دو چیز کنار هم یعنی concaténation.

مثال union

a+b

کلمه‌های accepted:

a, b

کلمهٔ ab accepted نیست، چون a+b فقط یک حرف می‌پذیرد.

مثال concaténation

ab

کلمهٔ accepted:

ab

کلمه‌های a و b accepted نیستند، چون باید هر دو پشت سر هم بیایند.

۱) Lemme d’Arden چیست؟

قانون Arden برای حل معادله‌های زبانی است. وقتی متغیر خودش دوباره در سمت راست ظاهر می‌شود، Arden کمک می‌کند آن را حذف کنیم و به expression rationnelle برسیم.

فرم اصلی:

X = AX + B

جواب:

X = A*B

معنی ساده: اول می‌توانیم A را صفر یا چند بار تکرار کنیم، بعد در آخر B را بخوانیم.
معادلهبا Ardenزبان
X = aX + bX = a*bچندتا a، بعد b
X = bX + aX = b*aچندتا b، بعد a
X = (a+b)X + abX = (a+b)*abهر چیزی، بعد ab

۲) چرا X = aX + b می‌شود a*b؟

معادله:

X = aX + b

یعنی برای ساختن یک کلمه از X دو راه داریم:

  • یا مستقیماً b را بگیریم.
  • یا اول یک a بگیریم و دوباره وارد X شویم.
اگر راه دوم را صفر بار انجام بدهیم:

b

اگر راه دوم را یک بار انجام بدهیم:

ab

اگر راه دوم را دو بار انجام بدهیم:

aab

پس زبان می‌شود:

{b, ab, aab, aaab, ...}

این دقیقاً:

a*b

است.

۳) فرم‌های Arden که در امتحان می‌بینی

فرمچطور تبدیل کنیم؟جواب
X = AX + B مستقیم Arden X = A*B
X = B + AX اول جابه‌جا کن: X = AX + B X = A*B
X = AX + BX + C مشترک بگیر: X = (A+B)X + C X = (A+B)*C
X = ε + aX جابه‌جا: X = aX + ε X = a*
X = aX اینجا B=∅ X = ∅ در جواب کمینه
در نسخه رسمی، برای یکتایی جواب معمولاً باید ε ∉ A باشد. در تمرین‌های ما معمولاً همین حالت است. فعلاً برای امتحان عملی کافی است فرم را درست تشخیص بدهی.

مثال ۴

اینجا A=aa و B=b

پس:

X = (aa)*b

۹) قالب امتحانی آماده

On utilise le lemme d'Arden. Si une équation est de la forme X = AX + B, alors la solution est X = A*B, à condition que ε ∉ A. On regroupe d'abord tous les termes contenant X, puis on applique le lemme pour éliminer la récursion.
