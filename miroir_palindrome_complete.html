Miroir et Palindrome — درس کامل

این فایل فقط موضوع miroir و palindrome را درس می‌دهد. بخش langage inductif را بعداً در همین سبک جدا می‌خوانیم.

۱) معنی miroir یک کلمه

miroir یعنی کلمه را برعکس کنیم. در فرانسوی معمولاً می‌نویسند:

˜w یا w~

یعنی miroir de w.

مثال ساده

قانون خیلی ساده

آخرین حرف می‌آید اول، حرف اول می‌رود آخر.

w = a b b a b

w~ = b a b b a

۲) تعریف بازگشتی miroir

Définition récursive du miroir :

ε~ = ε
Pour tout mot u et toute lettre c ∈ A :
(uc)~ = c u~
یعنی اگر یک کلمه به شکل uc باشد، یعنی آخرش حرف c است، در miroir آن، حرف c می‌آید اول، بعد miroir کلمهٔ قبلی u.

مثال با تعریف بازگشتی

می‌خواهیم miroir کلمهٔ abb را حساب کنیم.

کلمه را به شکل u c می‌نویسیم:

abb = ab . b

پس:

(abb)~ = b (ab)~

حالا:

ab = a . b

پس:

(ab)~ = b (a)~

و:

(a)~ = a

پس:

(ab)~ = ba

در نتیجه:

(abb)~ = bba

۳) قانون مهم: miroir de concaténation

قانون بسیار مهم برای امتحان:

(uv)~ = v~ u~

یعنی miroirِ concaténation ترتیب را برعکس می‌کند.

مثال

فرض کن:

u = ab

v = baa

پس:

uv = abbaa

و:

(uv)~ = aabba

حالا با قانون

v~ = aab

u~ = ba

پس:

v~u~ = aab ba = aabba

همان نتیجه است.

اشتباه رایج:

(uv)~ ≠ u~v~

درست:

(uv)~ = v~u~

۶) Miroir d’un langage

تا اینجا miroir یک کلمه را دیدیم. حالا اگر یک زبان L داشته باشیم، miroir زبان یعنی miroir همهٔ کلمات آن زبان.

تعریف:

L~ = { w~ | w ∈ L }

مثال ساده

اگر:

L = {ab, aab, b}

پس:

L~ = {ba, baa, b}

مثال با expression rationnelle

LL~توضیح
A*abbaA*کلمات ending with ab بعد از miroir می‌شوند starting with ba.
a*b*b*a*اول aها بعد bها، برعکس می‌شود.
(ab)*(ba)*هر بلوک ab برعکس می‌شود ba.
aA*bbA*aشروع با a و پایان با b برعکس می‌شود.
برای expression، فقط همه چیز را برعکس نکن؛ باید ترتیب concaténation را برعکس کنی:

(EF)~ = F~E~

۸) آیا زبان palindrome reconnaissable است؟

اگر الفبا فقط یک حرف داشته باشد، مثل A={a}، همهٔ کلمات palindrome هستند. پس recognisable است.
اما روی الفبای {a,b}، زبان همهٔ palindromeها معمولاً non reconnaissable است، چون اتومات finite باید نصف اول کلمه را حفظ کند تا با نصف دوم مقایسه کند، و حافظهٔ نامحدود ندارد.

برای امتحان تو معمولاً لازم نیست اثبات کامل non reconnaissable بودن palindrome را انجام بدهی، مگر اینکه سؤال دقیقاً بخواهد. بیشتر وقت‌ها تعریف، miroir، و چند propriété می‌آید.

۹) نوع سؤال‌های امتحانی درباره miroir

Types de questions possibles :
  1. Calculer le miroir d’un mot.
  2. Dire si un mot est un palindrome.
  3. Montrer que (uv)~ = v~u~.
  4. Calculer le miroir d’un langage ou d’une expression rationnelle.
  5. Construire un automate pour le miroir d’un langage reconnaissable.

۱۰) اگر بخواهیم automate du miroir بسازیم

اگر یک زبان L با یک automate شناخته شود، زبان miroir آن L~ هم reconnaissable است. ایده: همهٔ فلش‌های اتومات را برعکس می‌کنیم.

روش ساخت

  1. همهٔ transitions را برعکس کن.
  2. états finaux قدیمی تبدیل به états initiaux جدید می‌شوند.
  3. état initial قدیمی تبدیل به état final جدید می‌شود.
  4. اگر چند initial جدید داشتی، می‌توانی یک initial تازه اضافه کنی و با ε به آن‌ها وصل کنی.

اتومات اصلی برای ab

اتومات miroir برای ba

چرا؟ اتومات اصلی ab را می‌خواند. اگر فلش‌ها را برعکس کنیم، اتومات جدید ba را می‌خواند، یعنی miroir زبان قبلی.
