بخش ۱ - معنی (E, ≤) و (E, ⪯)

وقتی در صورت سؤال می‌بینی:

(E, ≤)

یعنی مجموعهٔ E را با یک رابطهٔ ترتیب بررسی می‌کنیم. اگر علامت ≤ باشد، معمولاً order معمولی است. اگر علامت ⪯ باشد، یعنی order عمومی است و خود سؤال معنی آن را تعریف می‌کند.

Soit (E, ⪯) un ensemble ordonné.

ترجمه: فرض کنیم E یک مجموعه است و ⪯ روی آن یک رابطهٔ order است.

در امتحان، هر وقت این جمله را دیدی، دیگر لازم نیست ثابت کنی order است؛ سؤال از قبل گفته ordered است.

بخش ۲ - علامت ≤

≤ یعنی کوچک‌تر یا مساوی. مساوی بودن هم قبول است.

2 ≤ 5   vrai
3 ≤ 3   vrai
5 ≤ 2   faux

در TD2، اگر روی اعداد باشد، از propriétés طبیعی استفاده می‌کنیم: réflexive, antisymétrique, transitive.

بخش ۳ - علامت <

< یعنی strict: کوچک‌تر واقعی، نه مساوی.

2 < 5   vrai
3 < 3   faux

فرق با ≤:

x < y  ⇔  x ≤ y et x ≠ y

در order عمومی هم نسخهٔ strict داریم:

x ≺ y  ⇔  x ⪯ y et x ≠ y

بخش ۴ - علامت‌های عمومی ≼ و ⪯

این دو علامت تقریباً همان معنی را دارند: یک relation d’ordre عمومی. معنی دقیق را خود سؤال می‌گوید.

x ⪯ y

یعنی: x نسبت به order سؤال، پایین‌تر یا مساوی y است.

n | m si et seulement si n est un diviseur de m.

اینجا n | m نقش order دارد: n پایین‌تر از m است اگر n عدد m را تقسیم کند.

بخش ۵ - علامت ⊑

⊑ هم یک علامت order است، ولی بیشتر وقتی می‌آید که order روی توابع، کلمات، یا ساختارها تعریف شده باشد.

f ⊑ g ssi pour tout n ∈ N, f(n) ≤ g(n)

این یعنی تابع f از تابع g کوچک‌تر است اگر برای همهٔ n مقدار f(n) از g(n) کوچک‌تر مساوی باشد.

در annale 2024 دقیقاً چنین سؤالی آمده است.

بخش ۶ - جمله‌های x ≤ y، x < y، x ⪯ y، x ≺ y

علامتمعنی فارسیفرانسوی امتحانی
x ≤ yx کوچک‌تر یا مساوی y استx est inférieur ou égal à y
x < yx واقعاً کوچک‌تر از y استx est strictement inférieur à y
x ⪯ yx طبق order سؤال پایین‌تر یا مساوی y استx est inférieur ou égal à y pour cet ordre
x ≺ yx طبق order سؤال strict پایین‌تر از y استx est strictement inférieur à y pour cet ordre

بخش ۷ - relation d’ordre یعنی چه؟

یک رابطه order است اگر دقیقاً این ۳ شرط را داشته باشد:

relation d’ordre = réflexive + antisymétrique + transitive

۱) réflexive

∀x ∈ E, x ⪯ x

هر عنصر با خودش رابطه دارد.

۲) antisymétrique

∀x,y ∈ E, (x ⪯ y et y ⪯ x) ⇒ x = y

دو عنصر متفاوت نمی‌توانند همزمان از دو طرف پایین‌تر از هم باشند.

۳) transitive

∀x,y,z ∈ E, (x ⪯ y et y ⪯ z) ⇒ x ⪯ z

اگر x پایین‌تر از y و y پایین‌تر از z باشد، پس x پایین‌تر از z است.

قالب جواب امتحان: برای نشان دادن order، همیشه سه پاراگراف بنویس: Réflexivité / Antisymétrie / Transitivité / Conclusion.

TD2 - Exercice 3

Énoncé exact du TD
Exercice 3
On définit la relation ⊆ IN × IN par (a, b)  (c, d) ssi a + b < c + d ou (a, b) = (c, d).
      1. Montrer que  est une relation d’ordre.
      2. Cet ordre est-il total ? bien fondé ? Justifier les réponses.
      3. Soit l’ensemble A = {(0, 2), (2, 1), (1, 2), (2, 2)}. Déterminer, s’ils existent, les minorants, les
         majorants, les éléments minimaux, les éléments maximaux, la borne inférieure, la borne su-
         périeure, le minimum, le maximum de A.
      4. Même question pour B = {(0, 1), (1, 0), (1, 1), (1, 2), (2, 1), (3, 0)}.
Correction / méthode
1) La relation est un ordre :
- réflexive : (a,b) = (a,b).
- antisymétrique : si a+b < c+d et c+d < a+b impossible; donc si les deux sens sont vrais, les couples sont égaux.
- transitive : si x ≼ y et y ≼ z, alors soit égalités, soit somme(x) < somme(y) < somme(z), donc x ≼ z.

2) Pas total : (0,2) et (1,1) ont même somme mais ne sont pas égaux, donc incomparables. Bien fondé : une descente stricte fait strictement baisser a+b, impossible infiniment dans N.

3) A = {(0,2),(2,1),(1,2),(2,2)}
minorants = {(0,0),(1,0),(0,1),(0,2)}
majorants = {(2,2)} ∪ {(p,q) ∈ N×N | p+q > 4}
élements minimaux = {(0,2)}
élements maximaux = {(2,2)}
inf = (0,2), sup = (2,2), minimum = (0,2), maximum = (2,2).

4) B = {(0,1),(1,0),(1,1),(1,2),(2,1),(3,0)}
minorants = {(0,0)}
majorants = {(p,q) ∈ N×N | p+q > 3}
élements minimaux = {(0,1),(1,0)}
élements maximaux = {(1,2),(2,1),(3,0)}
inf = (0,0), sup = n’existe pas, minimum = n’existe pas, maximum = n’existe pas.

TD2 - Exercice 13

Énoncé exact du TD
Exercice 13
On considère un ensemble E muni d’une opération binaire notée t telle que t est commutative,
associative et idempotente (c’est-à-dire pour tout x ∈ E, x t x = x). On définit la relation  sur E
par : x  y ssi x t y = y.
      1. Montrer que  est une relation d’ordre.
      2. Montrer que toute paire d’éléments admet une borne supérieure.
Correction / méthode
1) x ⪯ y ssi x ⊔ y = y.
Réflexive : x ⊔ x = x par idempotence.
Antisymétrique : si x⊔y=y et y⊔x=x, par commutativité x⊔y=y⊔x, donc y=x.
Transitive : si x⊔y=y et y⊔z=z, alors x⊔z = x⊔(y⊔z) = (x⊔y)⊔z = y⊔z = z.

2) Pour {x,y}, la borne supérieure est x ⊔ y.

TD2 - Exercice 15

Énoncé exact du TD
Exercice 15
Soit E un ensemble. On considère l’ensemble ℘(E) des parties de E et X un sous-ensemble de E.
      1. La relation ≤X définie sur ℘(E) par :

                                                   A1 ≤X A2 ssi A1 ∩ X ⊆ A2 ∩ X

          est-elle une relation d’ordre ? Pourquoi ?
      2. Montrer que la relation ≡X définie sur ℘(E) par :

                                                   A1 ≡X A2 ssi A1 ∩ X = A2 ∩ X

          est une relation d’équivalence.
      3. On considère l’ensemble des classes d’équivalence de ℘(E) pour la relation ≡X et on définit
         sur cet ensemble la relation X par :

                                                   [A1 ]≡X X [A2 ]≡X ssi A1 ≤X A2

          Montrer que X est une relation d’ordre.
Correction / méthode
1) ≤X n’est pas forcément antisymétrique sur P(E). Exemple : E={1,2}, X={1}, A={1}, B={1,2}. Alors A∩X=B∩X={1}, donc A≤X B et B≤X A, mais A≠B. Ce n’est pas un ordre.
2) ≡X est une équivalence : réflexive, symétrique, transitive par égalité de A∩X.
3) Sur les classes d’équivalence, la relation devient un ordre car l’antisymétrie est vraie au niveau des classes : si [A]≤[B] et [B]≤[A], alors A∩X=B∩X, donc [A]=[B].

بخش ۸ - ordre total / ordre partiel / comparable / Hasse

ordre total

∀x,y ∈ E, x ⪯ y ou y ⪯ x

هر دو عنصر قابل مقایسه‌اند.

ordre partiel

بعضی عناصر قابل مقایسه نیستند.

x et y incomparables ⇔ non(x ⪯ y) et non(y ⪯ x)

Diagramme de Hasse

در Hasse diagram، loop و فلش‌های transitivité را نمی‌کشیم. فقط رابطهٔ مستقیم را نشان می‌دهیم.

Relation d’ordre = réflexive + antisymétrique + transitive

P(E), ⊆: inf = intersection, sup = union
